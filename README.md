# WeatherNow hoàn chỉnh

## Chạy dự án

```bash
npm install
npm run dev
```

Mở http://localhost:3000

## Tính năng

- Thời tiết thật theo tọa độ bằng Open-Meteo
- AQI, PM2.5, PM10, CO, NO2, SO2, O3
- Chỉ số UV và cảnh báo
- Dự báo theo giờ và 7 ngày
- Bản đồ Leaflet: phố, địa hình, nền tối; nhấn để chọn vị trí
- GPS, tìm địa điểm, địa điểm yêu thích
- Lịch âm, ngày giờ, sáng/tối, Việt/Anh
- Đổi °C/°F và km/h/m/s
- Giao diện responsive desktop/mobile

Các component giao diện mẫu cũ đã được loại bỏ để tránh trùng logic và dữ liệu giả.
